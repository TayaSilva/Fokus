# Projeto FOKUS – Timer de Foco

Projeto simples desenvolvido com **HTML, CSS e JavaScript puro**, com foco em praticar manipulação do DOM, controle de estado e lógica de temporizador no estilo **Pomodoro**.

O objetivo é ajudar o usuário a alternar entre momentos de **foco**, **descanso curto** e **descanso longo**, com mudanças visuais, sons e contagem regressiva.

---

## Funcionalidades

- Contagem regressiva de tempo
-  Alternância entre modos:
  - Foco
  - Descanso curto
  - Descanso longo
- Mudança dinâmica de layout conforme o contexto
- Sons de iniciar, pausar e finalizar
- Música ambiente opcional
- Botão iniciar / pausar com feedback visual
